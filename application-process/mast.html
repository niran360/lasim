Mast Permit Application

Apply for Mast permits for telecommunications infrastructure deployment across Lagos State.

Note: Starting February 2026, all Mast applications will transition to the TIRS digital platform.

Application Process Timeline

1

Pre-Application Consultation

Consult with our technical team to understand requirements specific to your proposed route.

2

Document Preparation

Gather required documents including technical specifications, route maps, and corporate documents.

3

Application Submission

Submit complete application package through our current system (TIRS from Feb 2026).

TIRS Benefits for ROW Applications

With the Telecommunication Infrastructure Regulatory System launching in February 2026:

  • Processing time reduced from 30 days to 5 working days
  • Real-time application tracking dashboard
  • Automated GIS mapping integration
  • Digital document submission and verification
  • Instant notification system
